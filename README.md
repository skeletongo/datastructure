> + 是一种平衡二叉树
> + 引入平衡因子的概念：当前节点的平衡因子大小 = 当前节点左子树的高度 - 当前节点右子树的高度
> + 所有节点的平衡因子大小的绝对值要小于等于1
> + 四种需要维护节点平衡性的情况：LL,RR,LR,RL
